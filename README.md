
# Portfolio:
React 로 제작된 포트폴리오입니다.

## 🖥️ 프로젝트 소개
이 프로젝트는 포트폴리오입니다 위해 만들어진 프로젝트 입니다.

## ⏱️ 개발 기간
24년 01월 18일

## ⚙️ 기술 스택
- **React**: 웹 프론트엔드 라이브러리
- **Axios**: HTTP 요청을 위한 라이브러리
- **Bootstrap**: CSS 프레임워크로, 빠른 스타일링을 위해 사용
- **JavaScript (ES6+)**: 프로그래밍 언어
- **JSON**: 데이터 포맷으로 사용
